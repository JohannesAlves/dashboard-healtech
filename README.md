## Instalação

Para a instalação e execução do projeto prossiga da seguinte forma:
Utilizando os 3 comandos você irá conseguir subir o json-server, e iniciar o projeto.

```bash
  yarn
  yarn server
  yarn dev
```

## Rodando os testes

Para rodar os testes, rode o seguinte comando

```bash
  yarn test
```
